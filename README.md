# webpack2-seed

A seed project to start working with webpack 2 bundler

Includes:

 - SCSS
 - Dev Server
 - Bundler
 
## Get started

To get started:

```bash
git clone <this project>
cd <project directory>
npm i
```
To start the server:

```bash
npm start
```

To bundle:

```bash
npm build
```

To automatically push to gh-pages:

```bash
npm version <major|minor|patch> # follow the subversioning system
```
